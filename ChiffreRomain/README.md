# TP Chiffre Romains

Ecrire une fonction de conversion d'un chiffre de 1 à 50 en son équivalent Romain

Pour vérifier le non fonctionnement de la fonction, nous utiliserons les tests unitaires.

## Rappel

1 => I

4 => IV

5 => V

9 => IX

10 => X

40 => XL

50 => L

90 => XC

100 => C

400 => CD

500 => D

900 => CM

1000 => M
